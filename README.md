# ☕ Coffee Shop Sales Analysis (Excel Project)

## 📌 Project Overview
This project focuses on analyzing coffee shop sales data using Microsoft Excel. The goal is to extract meaningful insights, identify trends, and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling products
- Understand customer purchasing patterns
- Track revenue trends over time
- Create an interactive dashboard

---

## 🛠️ Tools & Technologies
- Microsoft Excel
  - Pivot Tables
  - Charts & Graphs
  - Data Cleaning
  - Conditional Formatting
  - Dashboard Design

---

## 📊 Dataset Description
The dataset includes:
- Order Date
- Product Name
- Category
- Quantity Sold
- Price
- Total Sales

---

## 🔍 Key Analysis Performed
- ✔️ Data Cleaning and Preprocessing
- ✔️ Sales Trend Analysis (Daily/Monthly)
- ✔️ Top Products Identification
- ✔️ Category-wise Sales Distribution
- ✔️ Revenue Analysis
- ✔️ Dashboard Creation

---

## 📈 Dashboard Features
- Sales overview (Total Revenue, Orders, Quantity)
- Top-selling products
- Category-wise performance
- Time-based sales trends
- Interactive filters (if applied)

---

## 💡 Key Insights
- Identified best-performing products and categories
- Found peak sales periods
- Observed customer buying behavior patterns
- Highlighted opportunities to increase sales
